# MCP Infrastructure Deployment Repository

This repository contains Terraform-generated infrastructure-as-code for cloud deployments.

## About

This repository is automatically populated with Terraform code generated through the MCP (Model Context Protocol) Infrastructure system.

## Usage

All infrastructure is defined using Terraform. Pull requests contain the generated Terraform code for review and approval before deployment.

## Contributing

Infrastructure changes should be submitted through the MCP Infrastructure system, which will automatically:
1. Generate appropriate Terraform code
2. Create a pull request with the code
3. Allow team review and approval
4. Deploy upon merge
